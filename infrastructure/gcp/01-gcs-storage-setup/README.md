# Google Cloud Storage (GCS) Setup

This module contains scripts and instructions for automating the creation of Google Cloud Storage buckets and managing model artifacts using the `google-cloud-storage` Python SDK.

## Prerequisites
- Authenticate using `gcloud auth application-default login` or set the `GOOGLE_APPLICATION_CREDENTIALS` environment variable.
- Install the required library:
  ```bash
  pip install google-cloud-storage
  ```
